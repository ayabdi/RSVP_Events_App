## Getting Started


### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

  1. Clone the repo
     ```sh
     git clone https://github.com/ayabdi/RSVP_Events_App.git
     ```
  2. Install NPM packages
     ```sh
      npm install
      ```
   
   ## Usage
   Link to Web app https://rsvp-events-app.vercel.app/
